# 電子計算機研究会会則置き場

岡山大学電子計算機研究会の会則・規則を管理するリポジトリ

> **一般部員の方へ**: 現在施行されている会則を確認したい場合は、**画面右側 `release`** のzipファイル内の`電子計算機研究会会則.pdf`をご覧ください

## mainについて

`main`ブランチは幹部によるレビュー済み(幹部会による前審査に若干対応)ですが，可決・施行前の情報を含みます
